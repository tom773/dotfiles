# Dotfiles

This repository contains my dotfiles. I use [GNU Stow](https://www.gnu.org/software/stow/) to manage them.

## Dependencies

### Git

```sudo apt install git
```

### stow

```sudo apt install stow
```

## Installation

``` $git clone git@github.com/tom773/dotfiles.git 
    $cd ~/.dotfiles
```
then

``` $stow .
```
